
# Car Insurance Claim Prediction

## Objective
Predict whether a policyholder will file a car insurance claim using ML models.

## Models Trained
- Logistic Regression
- Decision Tree
- Random Forest (Selected Best)
- XGBoost

## Best Model
Random Forest performed best based on ROC-AUC and F1 score.

## How to Run
pip install -r requirements.txt
streamlit run app/app.py
